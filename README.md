***How to Use!***
1. 환경변수에 Gemini API KEY를 등록한다.
2. 환경변수명은 GEMINI_API_KEY 로 작성(Default)
3. pip install -r requirements.txt
- requirements.txt에 작성된 라이브러리 install
3. 서버를 띄운다.
- CMD: uvicorn main:app --reload(option)
4. http://example.com/llmquestion/your_question
- your_question 부분에 질문을 작성
